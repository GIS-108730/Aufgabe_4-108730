## Introduction to Pandas DataFrames – Replicating an article in Markdown
This project is part of the UNIGIS module "Grundlagen der Anwendungsentwicklung" and follows the content from Lessons 8 and 9. The task was to practice working with Jupyter Notebook, Markdown, and version control using Git, GitHub, and Visual Studio Code. For this exercise, an online article was recreated in Markdown and reproduced inside a Jupyter Notebook so that the code and results are easy to follow and clearly documented.

### Project Files  
- **GitHub Repository**: The repository is used for the version control in this excercise and is needed as the main project folder, which contains all the other files of this excercise. It also documents the workflow with GitHub, including staging, committing, and pushing changes.  
- **Pandas.ipynb**: The main notebook, which based on the recreated online article. It contains all code cells and the corresponding outputs.  
- **Pandas.html**: This File is located in a subfolder *dist* of the repository and is the HTML export of the notebook Pandas. It allows viewing the content in a web browser without using Jupyter.  
- **Pandas. webp**: It allows viewing the content in a web browser without using Jupyter.
